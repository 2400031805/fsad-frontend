# Student Mental Health Frontend

## Run Locally

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Features
- Responsive React single-page application
- Routing with React Router
- Login/Register flow with validation
- Resource library and self-help materials
- Counseling scheduling form
- Peer support group listings
- Admin dashboard placeholder for content management
